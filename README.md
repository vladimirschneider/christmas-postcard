# Frontend workflow

## Features

* webpack
* Babel
* Eslint
* PostCSS
* Handlebars

## Install

1. Install [Node.js](http://nodejs.org/)
2. Clone the git repo `git clone https://github.com/vladimirschneider/frontend-workflow.git`
3. Install dependencies `npm install`

## How to use

### Develop

Run `npm run start:dev` or `npm run start:dev-watch` for watch

#### Production

Run `npm run start:prod`
